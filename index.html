主标题: 交互式地图示例

副标题: 使用 Leaflet 创建地图

这是一个简单的示例，展示如何在网页上使用 Leaflet 创建一个交互式地图。以下是本示例的 JavaScript 代码：

// 初始化地图
var map = L.map('map').setView([51.505, -0.09], 13);
// 添加瓦片图层
L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    maxZoom: 19,
    attribution: '© OpenStreetMap'
}).addTo(map);
// 添加标记和弹出窗口
L.marker([51.505, -0.09]).addTo(map)
    .bindPopup('A pretty CSS3 popup.
 Easily customizable.')
    .openPopup();

查看上面的地图，了解 Leaflet 的基本功能！
